========================================================================
STEP 2: CREATE NEW PROJECT
========================================================================

1. Click "New Project" (big purple button)
2. Select "Deploy from GitHub repo"
3. Authorize Railway to access GitHub (if asked)
4. Find and select: "pranab112/test"
5. Click "Deploy Now"

Railway will start building automatically!

========================================================================
STEP 3: ADD POSTGRESQL DATABASE
========================================================================

1. Wait for initial deployment (~1 minute)

4. Wait for database to provision (~30 seconds)

DATABASE_URL is automatically linked to your app!
